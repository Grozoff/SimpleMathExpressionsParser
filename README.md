# Simple math expressions parser
### Что умеет:

- Математические операции: умножение, деление, сложение, вычитание.
- Приоритет операций. 
- Скобки, скобки в скобках.
- Дробные числа, разделителем могут быть как точки так и запятые.
- Возведение в степень.
- Остаток от деления.
